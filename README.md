# PassChanger-
PassChanger is a security-focused Discord bot designed for educational purposes to demonstrate how legitimate account recovery and password reset assistance can be implemented safely within Discord communities.
The bot streamlines the recovery process by guiding users to official account-recovery services for platforms such as Gmail, Discord, Microsoft, GitHub, Minecraft, Steam, Epic Games, and more. It provides clear instructions, security recommendations, and phishing awareness while ensuring that all sensitive actions occur only through the provider's official website.
To help protect communities, PassChanger includes anti-spam and verification features such as CAPTCHA support, configurable rate limits, and moderation tools that reduce automated abuse. The bot also warns users about phishing attempts and can help moderators remove messages that accidentally expose sensitive information.
Features
Secure /process command for guided account recovery
Supports multiple account providers
Official recovery instructions and security guidance
CAPTCHA verification to reduce bot abuse
Configurable anti-spam and rate-limiting system
Ephemeral responses for improved privacy
Phishing awareness and online safety tips
Multi-server support
Clean, modern Discord slash-command interface
Lightweight, fast, and easy to configure
Security First
PassChanger is built with user privacy as its highest priority.
The bot does not collect, store, transmit, or process passwords, one-time passwords (OTPs), authentication codes, recovery codes, session cookies, or any other sensitive credentials. Users complete all verification steps directly with the official service provider.
Educational Purpose
This project exists to demonstrate best practices for secure account-recovery workflows, Discord bot development, and community security. It highlights how support bots can improve user experience without requesting confidential information or bypassing provider security systems.
PassChanger is intended for learning, development, and security awareness. It is not affiliated with Google, Discord, Microsoft, GitHub, Mojang, Steam, Epic Games, or any other service provider
